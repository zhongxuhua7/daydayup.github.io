# 家庭成长奖励网页 - 项目文档

## 项目概述
这是一个面向家庭的轻量级成长激励工具，通过「星星奖励 + 任务打卡 + 数据统计」机制，帮助家长培养孩子良好习惯。完全基于浏览器本地运行，无需部署服务器，数据存储在用户设备本地。

## 技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式设计（响应式布局，移动端优先）
- **JavaScript (Vanilla)** - 原生JavaScript，无框架依赖
- **localStorage** - 本地数据存储

## 核心特性
1. **纯前端本地运行** - 无需服务器，离线可用
2. **数据本地存储** - 所有数据存储在浏览器localStorage中
3. **移动端优先** - 响应式设计，适配各种屏幕尺寸
4. **数据导出导入** - 支持JSON格式数据备份和恢复

## 核心功能模块

### 1. 摘要页（首页）
- **数据概览卡片**：展示剩余星星数、月度统计
- **快捷操作按钮**：添加奖励、星星兑换
- **分类统计**：按分类展示本月奖励次数
- **最近记录**：显示奖励/兑换/扣减记录

### 2. 任务页
- **日期选择器**：查看本周每日任务
- **任务进度条**：显示当日任务完成率
- **任务列表**：勾选完成任务并自动发放星星
- **新增任务**：支持自定义任务名称、分类、星级

### 3. 数据页
- **日历视图**：查看每月奖励/兑换记录
- **统计卡片**：月度奖励次数、兑换次数、星星数
- **偏好分析**：展示奖励/兑换类型占比

### 4. 设置页
- **基本信息**：修改孩子昵称、头像
- **奖励分类管理**：新增/编辑/删除分类
- **兑换心愿管理**：新增/编辑/删除心愿
- **任务模板管理**：新增/编辑/删除任务
- **数据管理**：导出、导入、重置数据

## 核心业务逻辑

### 星星流转规则
1. **奖励发放**：完成任务 → 勾选任务 → 自动发放对应星星
2. **星星兑换**：选择心愿 → 确认兑换 → 扣除对应星星
3. **星星扣减**：手动扣减 → 选择原因和数量 → 扣除星星
4. **规则限制**：剩余星星不足时无法兑换，星星数不低于0

### 任务完成逻辑
- 勾选任务 → 自动发放星星 → 生成奖励记录 → 更新统计数据
- 已完成任务不可取消（确保奖励的严肃性）
- 支持每日重复任务

### 数据统计
- 统计周期：自然月
- 支持月份切换查看历史数据
- 自动计算奖励/兑换偏好占比

## 注意事项
1. 所有数据仅存储在用户本地设备，不向外部服务器传输
2. 清除浏览器数据会丢失所有记录，建议定期导出备份
3. 本地存储有容量限制（约5-10MB），建议定期清理旧数据
4. 数据导出后请妥善保管，避免丢失

## 常见问题

### Q: 如何备份数据？
A: 进入「设置」→「数据管理」→「导出数据」，下载JSON文件保存到本地。

### Q: 如何恢复数据？
A: 进入「设置」→「数据管理」→「导入数据」，选择之前导出的JSON文件。

### Q: 清除浏览器数据会怎样？
A: 会丢失所有记录，建议定期导出备份。

### Q: 可以在多台设备使用吗？
A: 目前不支持自动同步，但可以通过导出/导入数据在不同设备间迁移。

### Q: 如何删除历史记录？
A: 目前不支持单独删除记录，可以通过「重置数据」清空所有数据。
